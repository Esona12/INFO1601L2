Simple Selectors

Simple Selectors are used to select elements by their tag,id,clas or wildcard(*). They can also be grouped together by using commas


Combinator Selectors

These selectors are used to select elements based on the relationships between them.


a > b : Any selected element b that's a direct child of selected element a. An example of this is selecting the main element.


a ~ b: Any selected element b that comes after a selected element a. An example of this is selecting paragraph elements after a previous paragraph element


a + b: Any selected element b thats a sibling of a selected element a/. An example of this is selecting any elements that comes after the H1 element


a b: Any selected element b which is a contained with a selected element a. An example of this is selecting any of the paragraph elements within the footer element
